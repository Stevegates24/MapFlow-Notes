# 🧠 MapFlow Notes + AI

A simple web app to **learn CMT mapping** and **analyze products using AI**.

---

## 🚀 What this does

* 📘 Teaches CMT mapping (Temu → Amazon)
* 🤖 Helps you decide: **IF / INF / INVALID**
* ⚡ Speeds up your mapping decisions

---

## 🔗 Open App

👉 [MapFlow Notes](https://stevegates24.github.io/MapFlow-Notes/)

---

## ✨ Features

* Clean, structured **learning notes**
* **Step-by-step mapping flow**
* Inexact logic (Brand, Color, Material, etc.)
* Product rules (Softlines, Hardlines, Consumables)
* 🤖 AI assistant with:

  * Status suggestion
  * Confidence score
  * Reasoning
  * Attribute comparison

---

## 🧠 How to Use

1. Learn basics from **Notes section**
2. Go to **AI Assistant**
3. Paste:

   * Temu product details
   * (Optional) Amazon product
4. Click **Analyze**
5. Review result → apply in mapping
